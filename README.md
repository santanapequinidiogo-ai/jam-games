# Safe Roads - GGJ Safe Driving Sim

Este é um simulador de direção responsável desenvolvido para a **Global Game Jam** sob o tema "Own the Road".

## 🚗 O Conceito
Em vez de focar na velocidade pura, o objetivo do jogador é "tomar posse da estrada" através da responsabilidade. O sucesso é medido pelo **Safe Score** (Pontuação Segura), que aumenta ao manter distâncias seguras e respeitar os limites.

## 🛠️ Mecânicas Implementadas
- **Física de Inércia**: O carro possui peso e responde de forma diferente dependendo da velocidade e do clima.
- **Sistema de Clima Dinâmico**: A chuva reduz o atrito lateral, tornando curvas e frenagens mais perigosas.
- **Feedback Educativo**: Colisões ou atropelamentos pausam o jogo com dicas técnicas reais sobre segurança viária.
- **Eventos Aleatórios**: Tráfego variável e pedestres atravessando a pista.
- **HUD Premium**: Interface minimalista com indicadores de velocidade, clima e score.

## 🚀 Como Executar
1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Use as **Setas do Teclado** para controlar o veículo.

## 💻 Estrutura do Código
- `index.html`: Estrutura básica e UI.
- `style.css`: Design responsivo e estética premium (Glassmorphism).
- `game.js`: Engine principal, física e lógica de eventos.
